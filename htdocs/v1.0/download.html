For a list of all released files (Xpertmud source tar.gz and binary releases) look at the Sourceforge Xpertmud Download page.

Xpertmud scripts:

  • sysinit.pl - The standard perl script which is parsed each time the perl interpreter is loaded (Statically compiled into the interpreter).
  • sysinit.py - The standard python script which is parsed each time the python interpreter is loaded (Statically compiled into the interpreter).
  • bt3030.pl - A nice battletech mud script. This works with version above 1.0beta2 I think (not really sure).
  • battletech.pl (v0.1) - The new battletech mud script, supporting map caching, contact caching, scrollable and zoomable maps and a lot more. This is under heavy developement. Needs version 3 (upcoming release, try the latest cvs code).
If you've written some script, please mail it either Manuel Klimek or Ernst Bachmann (we're in need for those).
